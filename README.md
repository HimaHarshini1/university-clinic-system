# University Clinic Appointments Management System
**CSCE 5350 – Group 17**

## Setup Instructions
1. Install XAMPP
2. Import `schema.sql` into phpMyAdmin
3. Place all PHP files in `htdocs/your-folder/`
4. Visit `http://localhost/your-folder/index.php`

## Tech Stack
- PHP 8.x + PDO
- MySQL 8.0
- Bootstrap 5
- Apache (XAMPP)
